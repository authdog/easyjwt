<Translate id="introduction.installation.node-js-installation-tip">
    easyjwt can be installed via a package manager such as `yarn` or `npm`.
</Translate>

### yarn

<CodeBlock className="language-shell">
    yarn add @authdog/easyjwt
</CodeBlock>


### npm

<CodeBlock className="language-shell">
    npm install @authdog/easyjwt
</CodeBlock>
