## How to install

:::note

nodeJS 12+ is recommended to use `@authdog/easyjwt` in good conditions.
You can download the version adapted to your system on nodejs website.

:::

<Link to='https://nodejs.org/en/download/'>
Go to nodeJS releases website
</Link>

<br />
<br />

easyjwt can be installed via a package manager.

### nodeJS

- via yarn: `yarn add @authdog/easyjwt`
- via npm: `npm install @authdog/easyjwt`
