# How to install easyjwt?

## Installing on nodejs


easyjwt is distributed as a library available publicly on [npm](https://www.npmjs.com).


:::note

nodeJS 12+ is recommended to use `@authdog/easyjwt` in good conditions.
You can download the version adapted to your system on nodejs website.

:::

<Link to='https://nodejs.org/en/download/'>
    See nodeJS releases
</Link>

<br />
<br />

easyjwt can be installed via a package manager such as `yarn` or `npm`.

### via yarn

<CodeBlock className="language-shell">
    yarn add @authdog/easyjwt
</CodeBlock>


### via npm

<CodeBlock className="language-shell">
    npm install @authdog/easyjwt
</CodeBlock>
